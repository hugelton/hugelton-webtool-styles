Shari

A lightweight CSS framework for web editor tools with 20+ components, dark mode, and Fibonacci spacing.

v1.0.0 ~12KB 20+ Components MIT License

Installation

The fastest way to get started. Just add this line to your HTML:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/hugelton/shari@main/dist/shari.min.css">

Or use unpkg:

<link rel="stylesheet" href="https://unpkg.com/shari@latest/dist/shari.min.css">

Install via npm:

npm install shari

Then import in your HTML:

<link rel="stylesheet" href="node_modules/shari/dist/shari.min.css">

Download the CSS file directly:

Download Minified Download Expanded

Then include it in your HTML:

<link rel="stylesheet" href="path/to/shari.min.css">

Quick Start

Here's a minimal HTML template to get started:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My Web MIDI App</title> <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/hugelton/shari@main/dist/shari.min.css"> </head> <body> <div class="p-3"> <h1>My MIDI Controller</h1> <div class="panel"> <button class="button success">Connect Device</button> </div> </div> </body> </html>

Fibonacci Spacing System

Based on the Fibonacci sequence: 0, 1, 1, 2, 3, 5, 8, 13 (in rem units)

CSS Variables

--fib-0: 0rem /* 0px */ --fib-1: 1rem /* 18px */ --fib-2: 1rem /* 18px */ --fib-3: 2rem /* 36px */ --fib-4: 3rem /* 54px */ --fib-5: 5rem /* 90px */ --fib-6: 8rem /* 144px */ --fib-7: 13rem /* 234px */

Inter Tight Subheading (H3)

Body text uses Inter Tight with 18px base size and 1.6 line height for optimal readability in web applications. The font family is loaded from Google Fonts CDN.

/* Typography Settings */ Base Size: 18px (1rem) Headings: Inter Tight (h2-h6), Clash Display (h1) Body: Inter Tight, line-height 1.6

Dark Mode Theme

Toggle dark mode using the data-theme attribute:

// Enable dark mode document.body.setAttribute('data-theme', 'dark'); // Disable dark mode document.body.removeAttribute('data-theme'); // Save preference localStorage.setItem('theme', 'dark');
